Supported Canon Scanners

  • Canon CR-80
  • Canon CR-180

Steps

  1. Go to canon website to download the driver
  2. To search for the scanner, type in exactly imageFORMULA CR-80 (replace 80 with version of scanner being installed)
  3. Click on ‘Drivers’ (see image below)
  4. Select ‘Ranger’ and run the driver (see image below)
  5. Plug the scanner in
