Supported Scanners

Steps

  1. Unplug the scanner
  2. Access the capture page and click on ‘Drivers’ or click on the driver in the drop down below
  3. Download the appropriate driver
  4. Create a DadeScan folder on the c drive
  5. Unzip the driver and save the file in the DadeScan directory
  6. Run the setup with administrative rights from the DadeScan directory
  7. Plug in the scanner

Use the exerciser in the drop down menu to test the driver. Be sure to close all Dade applications, including the Capture Server before running. Must be installed with Admin rights.
